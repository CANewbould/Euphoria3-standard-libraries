Euphoria Standard library: msgbox

Routines

The following routines are part of the Eu3.1.1 installation and deliver exactly the same functionality as those defined in Open Euphoria's standard library of the same name.

  • message_box

Utilise this routine and the associated constants by adding the following statement to your module: 

include std\win32\machine.e

Interface

Includes

include std/machine.e	-- for free

include std/os.e	-- for WIN32
