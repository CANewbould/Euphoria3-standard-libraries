Euphoria Standard library: graphics

Routines

The following routines are part of the Eu3.1.1 installation and deliver exactly the same functionality as those defined in Open Euphoria's standard library of the same name.

  • bk_color
  • get_position
  • scroll
  • text_color
  • wrap

Utilise these routines by adding the following statement to your module: 

include std/graphics.e

Interface

Routines

global procedure bk_color(color c)	-- sets the background color to c - text or graphics modes

global function get_position()	-- returns {line, column} of current cursor position

global procedure scroll(integer amount, positive_int top_line, positive_int bottom_line)	-- scrolls a region of text on the screen

global procedure text_color(color c)	-- sets the foreground text color to c - text or graphics modes

Add 16 to get blinking. 

global procedure wrap(boolean on)   -- determines whether text will wrap when hitting the rightmost column

on = 1: characters will wrap at end of long line

on = 0: lines will be truncated
